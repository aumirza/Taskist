# Taskist

Task management using Asp.net core


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform

## Tech Stack
- Asp.net core (mvc) full stack
- mysql db

## Lessons Learned

- Asp.net
- c sharp
- Entity framework

## Installation

Clone this repo

```bash
git clone git@github.com:aumirza/Taskist.git
```

With github cli

```bash
gh repo clone aumirza/Taskist
```
